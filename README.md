## eggjs Websocket demo 服务项目
### ws 服务
本工程用于设备控制的基础模型。
1	心跳包
2	控制
3	子站状态变化
4 登录

### 本地项目启动

```bash
如果未安装cnpm , 先安装cnpm环境
拉取base, 先设置注册内源服务地址 

本项目只是用于代码参考，无内源base请自行构建，或者删除对应的代码替代。

$ npm i ntms-nodejs-base@latest 
$ cnpm i
$ npm run debug
$ open http://localhost:7003
```

### 配置说明
本项目不依赖 缓存和数据库。
